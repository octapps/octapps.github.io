OctApps Reference Manual

OctApps provides various functions intended to aid research scientists who perform searches for continuous gravitational waves. In also provides various general-purpose functions which may be of broader interest to users of Octave. Please see the project homepage for further information on OctApps.

This manual provides a series of tutorials which illustrate some of the functionality of OctApps. It also provides indexes of OctApps functions organised by directory, as well as alphabetically.
