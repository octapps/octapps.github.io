1.2 Continuous-Wave Functions

The tutorials in this section illustrate OctApps functions specialised to research into continuous gravitational waves.

• DoFstatInjections()  Tutorial on DoFstatInjections(), which creates and searches for simulated continuous gravitational-wave signals.
• ComputeDopplerMetric()  Tutorial on ComputeDopplerMetric(), which computes the parameter-space metric for continuous gravitational-wave searches.
• LatticeMismatchHist()  Tutorial on LatticeMismatchHist(), which generates theoretical mismatch histograms.
• SensitivityDepthStackSlide()  Tutorial on SensitivityDepthStackSlide(), which estimates the sensitivity of a continuous gravitational-wave search.
• OptimalSolution4StackSlide_v2()  Tutorial on OptimalSolution4StackSlide_v2(), which finds the optimal setup of a continuous gravitational-wave search.
